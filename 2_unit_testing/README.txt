Readme2: Unit Testing

PRE-REQUIREMENTS
- Java Runtime Enviroment is installed
- A network connection is active

How to create a chatroom
1. Open command prompt
2. Run the class file ChatServer with a port number as its parameter
	example : java ChatServer 293

How to join a chatroom
1. Open command prompt
2. Run the class file ChatClient with an internet address and a port number as its parameter
	example : java ChatClient 192.169.1.1 293
