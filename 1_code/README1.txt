---- ChatClient ----
HOW TO USE:
Run the ChatServer class in the command line.
Example : java ChatServer 293

Make sure that the class ServerThread and EDCrypt are located in the same folder.

---- ChatClient ----
HOW TO USE:
Run the ChatClient class in the command line.
Example : java ChatClient localhost 293


Make sure that the class ClientThread and EDCrypt are located in the same folder.
